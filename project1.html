YouTube Comments Insight AI

A Streamlit-powered web app that analyzes YouTube video comments to uncover audience sentiment, extract key insights, and visualize what viewers are actually talking about. Built using Hugging Face Transformers, PyTorch, KeyBERT, and Streamlit.

Open Live App →

Key Features

  • No API key required — scrapes comments directly
  • Sentiment analysis using twitter-roberta-base-sentiment
  • Keyword extraction with KeyBERT
  • Word cloud visualization of comment themes
  • Interactive charts powered by Streamlit

Example Workflow

  • Paste any YouTube video URL
  • App scrapes the first 200+ comments
  • Model predicts sentiment distribution
  • KeyBERT extracts meaningful keywords
  • Word cloud highlights dominant topics

Technologies Used

Python PyTorch Hugging Face Transformers KeyBERT Streamlit
