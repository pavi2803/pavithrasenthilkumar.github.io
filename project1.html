YouTube Comments Insight AI

AI-powered sentiment analysis and insight extraction from YouTube comments

View Code on GitHub → Live Demo →

Overview

A Streamlit-powered web application that analyzes YouTube video comments to uncover audience sentiment, extract key insights, and visualize what viewers are actually talking about. Built using Hugging Face Transformers, PyTorch, KeyBERT, and Streamlit to provide actionable insights without requiring API keys.

Key Features

No API Key Required

Directly scrapes comments without YouTube API limitations

Sentiment Analysis

Uses twitter-roberta-base-sentiment model for accurate predictions

Keyword Extraction

KeyBERT identifies the most relevant topics from comments

Word Cloud Visualization

Visual representation of dominant themes and topics

How It Works

  1. 1
    Input Video URL

    User pastes any YouTube video URL into the app

  2. 2
    Scrape Comments

    App extracts 200+ comments from the video

  3. 3
    Analyze Sentiment

    RoBERTa model predicts positive, negative, or neutral sentiment

  4. 4
    Extract Keywords

    KeyBERT identifies meaningful keywords and topics

  5. 5
    Visualize Results

    Display charts, metrics, and word clouds
