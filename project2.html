Lung Cancer Classification using CT Scans

An end-to-end deep learning pipeline for classifying lung cancer using CT scan images. The project follows a modular ML engineering structure with components for data ingestion, preprocessing, model training, evaluation, and experiment tracking.

View GitHub Repository →

Key Features

  • Complete end-to-end ML pipeline design
  • Data ingestion, preprocessing, and validation modules
  • 3D CT scan–based model training and evaluation
  • Experiment tracking and reproducibility via DVC
  • Config-driven architecture using YAML
  • Modular and scalable code structure for extensions

Workflow

  • Project Initialization Template script auto-generates project folders and boilerplate files.
  • Configuration Management config.yaml for paths and API connections; params.yaml for hyperparameters.
  • Data Ingestion Handles loading CT scan images and preparing datasets.
  • Base Model Preparation Sets up the deep learning model architecture.
  • Model Training Trains on processed CT lung images.
  • Model Evaluation Evaluates performance using standard metrics.
  • Pipeline Orchestration All stages are connected through main.py.
  • DVC Integration dvc.yaml ensures reproducible experiments.

Tech Stack

Python DVC YAML Deep Learning CT Imaging

Getting Started

  • Clone the repository git clone https://github.com/pavi2803/Lung_Cancer_Classification_CT_Scan.git
  • Install dependencies pip install -r requirements.txt
  • Configure config.yaml and params.yaml
  • Initialize project structure: python Template.py
  • Run the pipeline: python main.py
