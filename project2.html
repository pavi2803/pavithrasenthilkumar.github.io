Lung Cancer Classification App

End-to-end deep learning pipeline for classifying lung cancer from CT scan images

View Code on GitHub → Live Demo →

Overview

An end-to-end deep learning pipeline for classifying lung cancer using CT scan images. The project follows a modular ML engineering structure with components for data ingestion, preprocessing, model training, evaluation, and experiment tracking using DVC for reproducibility.

Demo

Screenshots showing CT scan processing, model implementation and classification result.

Key Features

End-to-End Pipeline

Complete ML pipeline from data ingestion to model evaluation

3D CT Scan Processing

Handles 3D medical imaging data for accurate classification

DVC Integration

Experiment tracking and reproducibility using Data Version Control

Config-Driven Architecture

YAML-based configuration for easy experimentation and deployment

Pipeline Workflow

  1. 1
    Project Initialization

    Template script auto-generates project folders and boilerplate files

  2. 2
    Configuration Management

    config.yaml for paths and API connections; params.yaml for hyperparameters

  3. 3
    Data Ingestion

    Loads CT scan images and prepares datasets for training

  4. 4
    Model Training

    Trains deep learning model on processed CT lung images

  5. 5
    Model Evaluation

    Evaluates performance using standard medical imaging metrics

  6. 6
    DVC Tracking

    dvc.yaml ensures reproducible experiments and version control

Technologies Used

Python Deep Learning DVC YAML CT Imaging MLOps

Getting Started

  1. Clone the repository:
    git clone https://github.com/pavi2803/Lung_Cancer_Classification_CT_Scan.git
  2. Install dependencies:
    pip install -r requirements.txt
  3. Configure parameters:
    Edit config.yaml and params.yaml
  4. Initialize project structure:
    python Template.py
  5. Run the pipeline:
    python main.py
